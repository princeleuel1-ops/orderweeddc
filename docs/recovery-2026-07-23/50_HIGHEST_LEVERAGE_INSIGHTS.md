# 50 HIGHEST-LEVERAGE INSIGHTS

1. The top-level form is a harness, not one giant agent.
2. Hermes governs; it does not erase CANA, SiteMind or Zenith.
3. CANA owns durability, not business truth.
4. Zenith owns proof/authority/evolution, not product pages.
5. SiteMind owns institutional intelligence and evidence services, not unilateral production deployment.
6. ORDERWEEDDC is the canonical commercial brand and conversion surface.
7. DistrictWeed is the supporting authority and local-intelligence network.
8. One source-controlled multi-domain app; no cloned doorway spam.
9. The next winning move is tighter verified intelligence, not more generic agents.
10. D.C. license truth is a defensible wedge.
11. Every public claim needs source, timestamp, state, freshness and publication eligibility.
12. Missing information must remain missing; never fill it with guesses.
13. Screenshots and DOM snapshots make changing websites reproducible evidence.
14. Playwright is the deterministic base; agent browsers sit above it.
15. Temporal/Restate/DBOS handle durability; LangGraph handles agent state.
16. Start memory with Postgres + pgvector + provenance; add graph complexity only when needed.
17. Retrieval confidence is not factual confidence.
18. Every substantial task should leave a context receipt.
19. No worker should certify its own output.
20. Persistent supervisor, finite missions.
21. Every mission needs budget, authority, retry ceiling, kill rule and rollback.
22. Two non-improving cycles should trigger stagnation handling.
23. Three similar failed attempts require root-cause analysis, not another retry.
24. A passing test proves one check, not the whole project.
25. Synthetic benchmarks must never be described as customer outcomes.
26. Recovery claims require preserved source evidence, not just metadata.
27. File census, extraction, analysis, synthesis, runnable code and live integration are different proof levels.
28. Real visual analysis must come from pixels, not filenames or SHA substrings.
29. A dynamic query engine must actually search indexes; file existence is not retrieval proof.
30. Organic ranking and sponsored placement must be separate and transparent.
31. Paid deal visibility should not continue when the consumer deal is hidden.
32. Marketplace monetization becomes stronger when paired with merchant-owned conversion and attribution.
33. Ads are not automated until revenue/profit is reconciled.
34. External media spend defaults to zero without explicit approval.
35. Track model/provider/cost and whether the expense created reusable learning.
36. “Aggressive” means faster evidence, better quality and more disciplined experiments—not spam.
37. Competitor mechanisms can be studied; protected expression must not be copied.
38. Visual excellence requires permanent design laws and verification, not prompt adjectives.
39. ORDERWEEDDC’s white, colorful, premium direction is a protected decision.
40. Public UI should not become a generic card-heavy agent dashboard.
41. The Website Institutional Digital Twin should include roles, routines, incentives and paid/organic mechanics.
42. SiteMind can become a sellable evidence/visual QA product before the full platform is complete.
43. The first sellable module can be Local Answer Governor / website QA and visibility monitoring.
44. Retailers pay for outcomes, prevented failures and operational certainty—not “AI” alone.
45. The strongest early offer is verified visibility + site QA + regulatory/local-answer monitoring.
46. A local licensed-retailer registry is the shared data foundation for consumer and B2B products.
47. Champion–challenger promotion must preserve trust, accessibility, privacy and performance.
48. Rejected hypotheses are assets; preserve them to prevent repeated failure.
49. Every accepted mission should update the capability frontier and next mission.
50. The archive’s purpose is to preserve the strongest mind while exposing what still is not real.
